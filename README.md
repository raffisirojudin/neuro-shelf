## 📌 Tentang Neuro-Shelf
  Neuro-Shelf adalah *tools* produktivitas berbasis web yang memungkinkan pengguna untuk merancang diagram alur (*flowchart*) secara instan. Alat ini sangat cocok untuk *developer* yang ingin memetakan logika algoritma sebelum menulis kode, atau untuk pemetaan proses bisnis.

  ### ✨ Fitur Utama
  * **Drag-and-Drop Canvas:** Pengguna bisa menarik berbagai bentuk diagram (proses, keputusan, *start/end*) ke dalam kanvas.
  * **Smart Connectors:** Garis penghubung antar-kotak yang otomatis menyesuaikan posisi saat komponen digeser.
  * **Export Options:** Kemampuan untuk mengeksport hasil *flowchart* menjadi file gambar (PNG/SVG) atau format JSON.
  * **Custom Styling:** Mengubah warna, ukuran teks, dan ketebalan garis pada setiap komponen diagram.

  ### 🛠️ Tech Stack
  * **Core:** HTML5 Canvas / SVG, CSS3, JavaScript (Object-Oriented Programming).
